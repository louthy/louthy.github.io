State<S, T>

For computations which read and write to a shared state.

Monad values are functions from the state to a (value, state). The bound function is applied to the bound value, and both have access to the shared state.

Useful for: Passing state through a computation without having to add a State argument to functions.

DECLARATION
public delegate StateResult<S, T> State<S, T>(S state)
PARAMETERS
state
S
NAMESPACE
LanguageExt
RETURNS
StateResult<S, T>
A value that represents the bound value and the resulting shared state value.
