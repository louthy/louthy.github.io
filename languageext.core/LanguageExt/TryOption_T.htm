TryOption<T>

The TryOption monad captures exceptions and uses them to cancel the computation. Additional the return value from the computation is an Option<T>, which can also cancel the bound computation. This captures the complete set of real result you could get from a function:

* Success * Failure (exception) * No value (null equivalent)
