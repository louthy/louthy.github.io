class TestTimeSpec Source

Unit tests often need to run at a specified time, or sometimes need time to stand still This is a simple specification builder for Sys.Test.TimeIO that allows you to pass that requirement through the runtime.

Methods

method TestTimeSpec FixedFromNow () Source

Time never passes, it has a constant value of DateTime.Now (as it was set when you call this method)

method TestTimeSpec FixedFromSpecified (DateTime state) Source

Time never passes, it has a constant value of state

method TestTimeSpec RunningFromNow () Source

Time passes normally, this is a straight pass-through to DateTime.Now

method TestTimeSpec RunningFromSpecified (DateTime state) Source

Time passes normally, but it is set with an initial state and flows from there
