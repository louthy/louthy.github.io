class ErrorException Source #

Error exception This works in tandem with the Error struct to allow for nested errors that easily convert between each other

Unlike regular exceptions, ErrorException can be either exceptional or non-exceptional, i.e. it is either created from an exception or it isn't. This allows for expected errors to be represented without throwing exceptions.
