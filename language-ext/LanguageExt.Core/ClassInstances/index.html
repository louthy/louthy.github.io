class Class <A> Source

Finds the default class instance for a type. So for Eq<string> it finds EqString. The result is cached so there's only a one time hit per type to resolve.

Parameters

type param A

Fields

field string Name Source

field IReadOnlyCollection<Type> All Source

field Option<string> Error Source

field A Default Source

Default class instance. If this is null then one couldn't be found, or more that one was found.
