LanguageExt.Core

LanguageExt.Core ► Monads ► Alternative Value Monads ► Try

The Try monads are all exception catching monads. Their alternative value is Exeption. They're useful for wrapping up calls to non-functional exception-throwing methods and making everything declarative.

NOTE: These monads have been superseded by the Eff and Aff monads in the language-ext Effects System. They will continue to be supported, but no new functionality added
