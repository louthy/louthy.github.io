    property object? Case Source #

    Reference version for use in pattern-matching

    Empty collection     = result is null
Singleton collection = result is A
More                 = result is (A, Seq<A>) -- head and tail

    Example:

    var res = arr.Case switch
{

   A value         => ...,
   (var x, var xs) => ...,
   _               => ...
}
